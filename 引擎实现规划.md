
这是一个**游戏引擎开发路线图/版本更新日志**，记录了从2022年4月4日首次发布后的功能规划。按模块整理如下：

## 已实现 ✅
| 模块 | 功能 |
|------|------|
| **编辑器** | 加载/保存关卡、增删改/移动/旋转/缩放物体 |
| **图形渲染** | 编辑器内运行(PIE)、FBX格式支持、子网格、前向渲染 |
| **动画** | 动画混合、阴影 |
| **碰撞** | 球体、包围盒碰撞检测 |
| **角色/相机** | 第一/第三人称相机、八方向移动+冲刺 |
| **架构** | 单线程基于对象的Tick更新 |

## 待实现 ❌
| 模块 | 计划功能 |
|------|---------|
| **动画** | 简单骨骼动画 |
| **游戏玩法/脚本** | （未展开） |

| **核心架构** | 多线程框架、ECS实体组件系统、资源管理器、空间划分 |
