# 05 - 行列式

## 核心观点

> **行列式 = 线性变换对面积的缩放比例**
> > |det(M)| = 变换后面积 / 变换前面积
> > sign(det(M)) = 是否翻转了空间的方向

## 从几何角度看行列式

### 单位正方形的变化

```
变换前：                变换后：

    y                      y'
    ↑                      ↑
  1 ┤  ┌──┐              1'┤     ↗
    │  │  │                 │    /  
  0 ┼──┴──┼──→ x        0'┼───●'──→ x'
    0  1                    
    
单位正方形面积 = 1      变成平行四边形，面积 = det(M)
```

### 行列式的定义

对于 2×2 矩阵：

```
    │a b│
det │c d│ = ad - bc
    
几何意义：
• │ad - bc│ = 平行四边形的面积
• 符号 = 方向是否翻转
```

## 为什么 det(M) = ad - bc？

### 几何推导

```
基向量变换后：
î → [a]    ĵ → [b]
    [c]        [d]
    
平行四边形面积 = |î' × ĵ'| （二维叉积）
               = |ad - bc|
```

```
面积计算：

    [b]  
     ↗d┌──┐
   [a]│  │
    c└──┘
    
使用行列式公式：
面积 = 底 × 高（投影）= ad - bc
```

## 行列式的性质（几何解释）

### 1. 单位矩阵的行列式 = 1

```
det(I) = 1

几何：面积不变（缩放比例 = 1）
```

### 2. 如果两列相同，det = 0

```
    │a a│
det │c c│ = ac - ac = 0

几何：两个基向量共线，平行四边形"压扁"成线段，面积 = 0
```

### 3. det(AB) = det(A)det(B)

```
几何：连续两个变换的总面积缩放 = 各自缩放的乘积
```

### 4. 交换两列，det 变号

```
    │a b│         │b a│
det │c d│ = D  det │d c│ = -D

几何：交换基向量 = 翻转空间方向
```

## 行列式的符号：方向

### 定向（Orientation）

```
正定向（det > 0）：       负定向（det < 0）：

    y                      y
    ↑                      ↑
    │  ĵ                   │ ↗ ĵ'
    │ ↗                    │/
    ●──→ î                 ●──→ î'
    x                      x
    
î → ĵ 是逆时针            î' → ĵ' 是顺时针（翻转了！）
```

**三维**：右手系 → 左手系 表示方向翻转

## 行列式为零意味着什么？

### det(M) = 0 的几何意义

```
变换将空间"压扁"到低维：

• 二维 → 一维（直线）
• 三维 → 二维（平面）或一维
```

```
矩阵 M = [2 1]    det(M) = 0
         [4 2]
         
第2列 = 2 × 第1列（线性相关）

变换后：整个平面 → 一条直线
```

### 与可逆性的关系

```
det(M) ≠ 0  ⟺  M 可逆
            ⟺  变换不压扁空间
            ⟺  存在逆变换
            
det(M) = 0  ⟺  M 不可逆
            ⟺  信息丢失（多对一映射）
```

## 三维行列式

### 几何意义

```
3×3 矩阵的行列式 = 平行六面体的有向体积

        │a b c│
    det │d e f│ = 体积缩放比例
        │g h i│
```

```
单位立方体 ──→ 平行六面体
    体积 = 1      体积 = |det(M)|
```

### 计算公式（萨吕法则）

```
    │a b c│
det │d e f│ = a(ei - fh) - b(di - fg) + c(dh - eg)
    │g h i│

记忆：对角线法则（仅适用于 3×3）
```

## 行列式计算示例

### 例1：旋转矩阵

```
    │cosθ  -sinθ│
R = │sinθ   cosθ│

det(R) = cos²θ + sin²θ = 1

几何：旋转不改变面积，不翻转方向 ✓
```

### 例2：剪切矩阵

```
    │1  k│
S = │0  1│

det(S) = 1

几何：剪切保持面积不变（像推扑克牌）✓
```

### 例3：缩放矩阵

```
    │2  0│
S = │0  3│

det(S) = 6

几何：面积放大 6 倍（2×3）✓
```

### 例4：投影矩阵

```
    │1  0│
P = │0  0│

det(P) = 0

几何：压扁到 x 轴，面积 = 0 ✓
```

## 关键洞察

### 不要死记公式，记住几何

> **行列式 = 面积（或体积）缩放比例 × 方向**

这个直觉能帮你：
- 理解为什么 det(AB) = det(A)det(B)
- 记住公式（2×2: ad-bc）
- 判断矩阵是否可逆
- 理解特征值（下一集）

### 行列式的本质

| 维度 | det(M) 测量 | det=0 意味着 |
|------|------------|-------------|
| 2D | 面积缩放 | 变换成直线 |
| 3D | 体积缩放 | 变换成平面或直线 |
| nD | n维体积缩放 | 降维（信息丢失）|
