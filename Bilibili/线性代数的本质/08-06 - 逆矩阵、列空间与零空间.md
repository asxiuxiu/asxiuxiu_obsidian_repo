# 06 - 逆矩阵、列空间与零空间

## 核心问题

> 当线性变换"丢失信息"时会发生什么？> > 线性方程组 Ax = v 何时有解？解是否唯一？

## 逆矩阵（Inverse Matrix）

### 定义

如果矩阵 A 表示一个线性变换，则 **A⁻¹** 表示**撤销**该变换。

```
A⁻¹A = AA⁻¹ = I

几何：先做 A 变换，再做 A⁻¹ 变换 = 回到原点
```

### 逆矩阵存在的条件

```
A⁻¹ 存在 ⟺ det(A) ≠ 0
         ⟺ A 不把空间压扁到低维
         ⟺ 变换是"一一对应"的
```

### 几何直观

```
可逆的情况：              不可逆的情况：

    y                      y
    ↑                      ↑
    │  ●v                  │
    │ /                    │   一条线上
    ●──→        A⁻¹       ●─────→
       x                      x
       
每个点都有唯一的"原像"
   ↓                     
压缩到直线后，
无法确定来自哪里
```

## 列空间（Column Space）

### 定义

> **列空间 = 矩阵列向量的所有线性组合**
> > = 变换后的所有可能输出

```
给定矩阵 A = [c₁ c₂ c₃ ...]

Col(A) = Span{c₁, c₂, c₃, ...}
       = { A·x | 所有 x }
       = 所有可能的输出向量
```

### 几何意义

```
二维矩阵 A = [î' ĵ']

列空间是 î' 和 ĵ' 张成的空间：

• 如果 î', ĵ' 不共线 → 列空间 = 整个平面（2D）
• 如果 î', ĵ' 共线   → 列空间 = 一条直线（1D）
```

### 与方程组的关系

```
方程 Ax = v 有解  ⟺  v 在 A 的列空间中

几何：v 能被 A 的列向量线性表示
```

## 零空间（Null Space / Kernel）

### 定义

> **零空间 = 所有被变换到原点的输入**

```
Null(A) = { x | Ax = 0 }
        = 变换后"消失"的所有向量
```

### 几何意义

```
变换 A：

    y                      y
    ↑                      ↑
    │  v                   │
    │ ↗                    │    0
    ●──→ x      →          ●─────→
    
    v ≠ 0                  Av = 0
    
    v 在零空间中！
```

### 例子

```
投影到 x 轴：

    y                      y
    ↑    ●(2,3)            ↑
    │    │                 │
  3 ┤    │    投影       3 ┤
    │    ↓                 │
  0 ┼────┼──→          0 ┼────●(2,0)
    0    2                   0    2
    
任何 (0, y) 形式的向量 → 投影到原点
零空间 = y 轴（所有 x=0 的向量）
```

## 秩（Rank）

### 定义

> **秩 = 列空间的维度**

```
rank(A) = dim(Col(A))
        = 线性无关的列向量个数
        = 变换后空间的维度
```

### 满秩（Full Rank）

```
满秩：rank(A) = 矩阵的行数/列数（取较小者）

• 2×2 矩阵，rank = 2 → 满秩，可逆
• 2×3 矩阵，rank = 2 → 满秩（最大只能是2）
• 3×3 矩阵，rank = 2 → 不满秩，不可逆
```

### 秩与零空间的关系

**秩-零化度定理（Rank-Nullity Theorem）**：

```
对于 n×n 矩阵 A：

rank(A) + dim(Null(A)) = n

（列空间维度）+（零空间维度）= 输入空间维度
```

```
直观理解：

输入空间 ℝⁿ ──→ 输出空间
    │              │
    ├─→ 列空间（被映射到的部分）
    │      rank(A) 维
    │
    └─→ 零空间（被压缩到0的部分）
           nullity(A) 维
           
两者维度之和 = n
```

## 线性方程组的解

### 情况分析

给定 Ax = v，设 A 是 n×n 矩阵：

| 情况 | 条件 | 解的个数 | 几何解释 |
|------|------|---------|---------|
| 唯一解 | det(A) ≠ 0 | 1 | 可逆，一对一映射 |
| 无解 | v ∉ Col(A) | 0 | v 不在输出范围内 |
| 无穷解 | det(A) = 0 且 v ∈ Col(A) | ∞ | 多个输入映射到同一输出 |

### 无穷解的情况

```
当 A 不可逆时，如果 v 在列空间中：

特解 xₚ：Axₚ = v
零空间中任意向量 xₙ：Axₙ = 0

则 A(xₚ + xₙ) = Axₚ + Axₙ = v + 0 = v

通解 = 特解 + 零空间中的任意向量
```

## 关键洞察

### 三个概念的统一视角

```
                    输入空间 ℝⁿ
                         │
           ┌─────────────┼─────────────┐
           │             │             │
           ▼             ▼             ▼
      列空间           零空间        （其他）
    （输出范围）    （被压缩到0）
           │             │
           │             │
           └──────┬──────┘
                  │
            rank + nullity = n
```

### 记忆口诀

> - **列空间**：变换能到达哪里
> - **零空间**：什么被压没了
> - **秩**：还有多少维度
