# 02 - 线性组合、张成的空间与基

## 核心概念

### 线性组合（Linear Combination）

给定两个向量 **v** 和 **w**，它们的**线性组合**是：

```
a·v + b·w

其中 a, b 是任意实数（标量）
```

**几何意义**：通过拉伸/压缩 v 和 w，然后相加，能到达的所有点。

### 张成的空间（Span）

> **所有可能的线性组合构成的集合**

```
Span(v, w) = { a·v + b·w | a, b ∈ ℝ }
```

## 二维情况的分类

### 情况1：两向量不共线

```
       ↗ w
      /
     /    ↗
    ●─────→ v
    
    张成的空间 = 整个二维平面！
```

当你可以自由调节 a 和 b 时，**任何二维向量**都可以表示为 a·v + b·w。

### 情况2：两向量共线

```
    w →
        
    v →
        
    ●────────→
    
    张成的空间 = 一条直线
```

无论怎么组合，结果都落在同一直线上。

### 情况3：其中一个是零向量

```
    v = [0, 0]
    
    w →
    
    张成的空间 = 一条直线（沿w方向）
```

## 三维空间的直觉

### 两个向量的情况

```
        z
        │
        │   ↗ w
        │  /
        │ /  ↗
        ●─────→ v
       / 
      /   
     y    x
```

**Span(v, w) = 一个平面**（过原点的平面）

### 三个向量的情况

```
        z
        │
        │   ↗ u
        │  /
    w ──┼─→
       /│
      / │
     ●──┼──→ v
    /   │
   y    x
```

**如果 u 不在 Span(v, w) 所在的平面上** → Span(v, w, u) = **整个三维空间**

## 线性相关与线性无关

### 线性相关（Linearly Dependent）

> 一个向量可以表示为其他向量的线性组合

```
    u = 2v + 3w
    
    u 是"多余的" —— 添加 u 不会扩大张成的空间
```

**几何表现**：
- 二维：两向量共线
- 三维：三向量共面

### 线性无关（Linearly Independent）

> 每个向量都"贡献"了新的维度

```
    u ≠ a·v + b·w  （对任何 a, b）
    
    添加 u 后，张成的空间从平面变成三维空间
```

## 基（Basis）

### 定义

一组向量 **{v₁, v₂, ..., vₙ}** 是某空间的**基**，当且仅当：

1. **线性无关** —— 没有冗余
2. **张成该空间** —— 能到达空间中任意点

### 标准基

二维标准基：
- **î** = [1, 0]
- **ĵ** = [0, 1]

```
        ↑ ĵ
        │
        │
   ─────●────→ î
        │
        │
```

任意向量 [x, y] = x·î + y·ĵ

### 基的选择不是唯一的！

```
       ↗ b₂
      /
     / 
    ●────→ b₁
    
    {b₁, b₂} 也可以作为基
```

同一个向量，在不同基下有不同的坐标。

## 关键洞察

### 张成空间的大小

| 维度 | 1个向量 | 2个向量 | 3个向量 |
|------|---------|---------|---------|
| 2D | 直线或点 | 平面（如果无关）| 平面（有冗余）|
| 3D | 直线 | 平面 | 空间（如果无关）|

### 核心思想

> **线性代数的核心问题是：给定一组向量，它们张成的空间是什么？**

这决定了：
- 方程组是否有解
- 解的唯一性
- 空间的维度

## 思考问题

1. 为什么两个不共线的向量能张成整个二维平面？
2. 三维空间中，两个向量最多能张成什么？
3. 如果 n 个向量张成的空间维度小于 n，意味着什么？
4. 标准基的优势是什么？为什么它最常用？
