# 03 - 矩阵与线性变换

## 核心观点

> **矩阵 = 线性变换**
> > 矩阵的每一列，是变换后的基向量！

## 什么是变换？

**变换** = 一个函数，输入一个向量，输出另一个向量

```
输入 v ──→ [变换] ──→ 输出 T(v)
```

### 可视化：变换向量场

```
变换前：              变换后：

  │││                  \\│//
 ─┼┼─               ──\│/──
 ─┼┼┼─               ──/│\──
  │││                  //│\\
  
网格保持平行且等距     网格仍然是直线（但可能旋转/缩放）
```

## 线性变换的两条规则

变换 **T** 是线性的，当且仅当：

1. **保持加法**：T(v + w) = T(v) + T(w)
2. **保持数乘**：T(c·v) = c·T(v)

### 几何意义

```
变换前：    变换后：
   ↗         ↗'
  /          /
 ●────→     ●'───→'
 
 平行四边形 → 仍是平行四边形（可能变形）
```

**推论**：
- 原点保持不动
- 直线仍是直线
- 网格线保持平行且等距分布

## 如何用数值描述线性变换？

### 关键观察

只需知道 **î** 和 **ĵ** 变换到哪里，就能知道**所有向量**的变换结果！

为什么？因为任何向量都可以写成：
```
v = x·î + y·ĵ
```

根据线性性质：
```
T(v) = T(x·î + y·ĵ)
     = x·T(î) + y·T(ĵ)
```

### 矩阵的列 = 变换后的基向量

```
        │ ↑ │
    M = │ T(î)  T(ĵ) │
        │ ↓   ↓ │
        
第一列：î 变换后的位置
第二列：ĵ 变换后的位置
```

## 具体例子

### 例1：逆时针旋转 90°

```
变换前：î = [1]    ĵ = [0]
            [0]        [1]
            
变换后：î → [0]    ĵ → [-1]
            [1]        [ 0]
            
旋转矩阵：R = [ 0  -1]
             [ 1   0]
```

```
    y              x
    ↑              ↑
    │    旋转90°   │
    ●──→           ↓
    │              ●
```

### 例2：沿 x 轴剪切（Shear）

```
变换后：î → [1]    ĵ → [1]
            [0]        [1]
            
剪切矩阵：S = [1  1]
             [0  1]
```

```
剪切效果：

变换前：          变换后：
┌─┬─┬─┐          ┌╲│╱┐
├─┼─┼─┤    →     ├─│─┤
├─┼─┼─┤          ├─│─┤
└─┴─┴─┘          └╱│╲┘

ĝ 向 → 方向倾斜
```

### 例3：缩放（Scale）

```
沿 x 轴缩放 3 倍，沿 y 轴缩放 2 倍：

S = [3  0]
    [0  2]
```

## 矩阵-向量乘法 = 应用变换

```
    [a  b]   [x]   [ax + by]
    [c  d] × [y] = [cx + dy]
    
    = x·[a] + y·[b]
        [c]     [d]
        
    = x·(第一列) + y·(第二列)
```

**这就是线性组合！**

## 从几何角度理解矩阵乘法

```
给定矩阵 M = [î'  ĵ']  （î'和ĵ'是变换后的基向量）

M × [x] = x·î' + y·ĵ'
    [y]
    
新坐标 = 原坐标在新基下的线性组合
```

```
        y                    y'
        ↑                    ↑
        │   v=[2,1]          │
   [0,3]┼  ↗                 ┼    v' = 2î' + 1ĵ'
        │ /                  │   ↗
   ─────●───→ [3,0]   →     ─────●'──→
       /│ x                 ↗   │   x'
      / │                ↗     │
     î  ĵ              î'      ĵ'
```

## 关键洞察

### 矩阵列的几何意义

> **矩阵的列向量 = 变换后的标准基向量**

这意味着：
- 观察一个矩阵，你立即知道它会如何变换空间
- det(M) = 变换后单位面积的变化比例
- 矩阵可逆 ⟺ 变换不"压扁"空间

### 线性变换的例子

| 变换 | 矩阵形式 | 几何效果 |
|------|---------|---------|
| 旋转 θ 度 | [cosθ -sinθ]<br>[sinθ  cosθ] | 刚性旋转 |
| 缩放 | [sₓ  0]<br>[0  sᵧ] | 各轴独立缩放 |
| 剪切 | [1  k]<br>[0  1] | 一轴向另一轴倾斜 |
| 投影 | ? | 降维（不可逆）|
| 反射 | ? | 镜像翻转 |
