# 附注1 - 三维空间中的线性变换

## 核心观点

> 二维的直觉直接推广到三维：> > **3×3 矩阵的列 = 三维空间中变换后的 î, ĵ, k̂**

## 三维空间的基础

### 标准基向量

```
        z
        │
        │  k̂ = [0, 0, 1]
        │   ↗
        │  /
   ĵ ───┼─→
       /│    î = [1, 0, 0]
      / │
     ●──┴──→ x
    /    î
   y
   
ĵ = [0, 1, 0]
```

### 3×3 矩阵的结构

```
    │ ↑  ↑  ↑ │
M = │ î' ĵ' k̂'│
    │ ↓  ↓  ↓ │
    
每一列是一个三维向量！
```

## 三维变换的例子

### 1. 绕各坐标轴旋转

**绕 z 轴旋转（xy平面旋转）**：

```
        z                    z
        │                    │
        │  k̂不变             │  k̂' = k̂
        │                    │
   ĵ ───┼─→            ĵ' ───┼─→ î'
       /│                   /│
      / │      旋转θ       / │
     ●──┴──→ x      →     ●──┴──→
        î                     
        
Rz(θ) = [cosθ  -sinθ   0]
        [sinθ   cosθ   0]
        [  0      0    1]
```

**绕 x 轴旋转（yz平面旋转）**：

```
Rx(θ) = [1     0        0   ]
        [0   cosθ   -sinθ ]
        [0   sinθ    cosθ ]
```

**绕 y 轴旋转（xz平面旋转）**：

```
Ry(θ) = [ cosθ   0   sinθ ]
        [  0     1    0   ]
        [-sinθ   0   cosθ ]
```

### 2. 缩放变换

```
        [sx  0   0 ]
S =     [0   sy  0 ]
        [0   0   sz]

各轴独立缩放
```

### 3. 投影变换

**投影到 xy 平面**（z 坐标变为0）：

```
        [1  0  0]
Pxy =   [0  1  0]
        [0  0  0]
```

## 右手坐标系 vs 左手坐标系

```
右手系（本系列使用）：     左手系：

    z                        z
    ↑                        ↑
    │  y                     │  y
    │ ↗                      │ ↙
    ●──→ x                   ●──→ x
    
右手定则：x → y = z 方向
```

## 三维旋转的组合

### 万向节锁问题（Gimbal Lock）

```
欧拉角旋转顺序很重要！

常见顺序：ZYX（偏航-俯仰-翻滚）

当俯仰角 = 90° 时：
- 第一次旋转（Z）和第三次旋转（X）
- 绕同一条轴旋转！
- 失去一个自由度

这就是"万向节锁"
```

### 四元数简介

> 为了避免万向节锁，计算机图形学中常用**四元数（Quaternions）**表示旋转。

优点：
- 避免万向节锁
- 插值更平滑
- 计算更高效

## 从二维到三维的直觉迁移

| 概念 | 二维 | 三维 |
|------|------|------|
| 基向量 | î, ĵ | î, ĵ, k̂ |
| 矩阵大小 | 2×2 | 3×3 |
| 张成的空间 | 直线/平面 | 直线/平面/空间 |
| 旋转轴 | 点（原点） | 直线 |
| 行列式意义 | 面积缩放 | 体积缩放 |

## 关键洞察

### 三维变换的"列视角"依然成立

```
给定矩阵 M = [c₁ c₂ c₃]

M × [x] = x·c₁ + y·c₂ + z·c₃
    [y]
    [z]
```

**只要记住：列是变换后的基向量，一切推导都自然成立。**
