# 04 - 矩阵乘法与线性变换复合

## 核心观点

> **矩阵乘法 = 连续应用两个线性变换**
> > AB 表示：先做 B 变换，再做 A 变换

## 复合变换

### 连续应用两个变换

```
v ──→[变换M₂]──→ M₂v ──→[变换M₁]──→ M₁(M₂v)
```

**复合效果**：
```
M₁(M₂v) = (M₁M₂)v
```

### 几何示例：旋转 + 剪切

```
步骤1：旋转90°          步骤2：剪切

   y                      y
   ↑                      ↑
   │   [0,-1]             │  ↗ [1,1]
   │  ↗                   │ /
   ●──→[1,0]      →       ●──→[1,0]
      x                     x
```

先旋转再剪切 ≠ 先剪切再旋转

## 矩阵乘法的几何推导

### 关键问题

已知：
- M₁ 将 î → [a, c]，ĵ → [b, d]
- M₂ 将 î → [e, g]，ĵ → [f, h]

问：复合变换 M₁M₂ 是什么？

### 推导过程

**M₁M₂ 的第一列** = M₁ 作用于 M₂ 的第一列

```
M₂ 的第一列 = [e]  （这是 M₂ 变换后的 î）
             [g]

M₁ 作用于此：
M₁[e] = e·[a] + g·[b] = [ae + bg]
  [g]     [c]     [d]   [ce + dg]
```

**结果**：
```
        [ae+bg  af+bh]
M₁M₂ =  [ce+dg  cf+dh]
```

这就是矩阵乘法公式！

## 矩阵乘法的计算规则

```
[a b]   [e f]   [ae+bg  af+bh]
[c d] × [g h] = [ce+dg  cf+dh]

(行 from 左) · (列 from 右)
```

```
┌           ┐   ┌       ┐   ┌               ┐
│ 第1行     │   │ 第1列 │   │ 点积(第1行,第1列) │
│ · · ·     │ × │ · · · │ = │ · · ·           │
│ · · ·     │   │ · · · │   │ · · ·           │
└           ┘   └       ┘   └               ┘
```

## 为什么矩阵乘法不满足交换律？

### 几何解释

变换顺序很重要！

```
例：先旋转90°，再剪切

原始：    旋转后：   剪切后：
┌─┬─┐     ┌╲┬╱┐     ┌╲│╱┐
├─┼─┤  →  ├─┼─┤  →  ├─│─┤
└─┴─┘     └╱┴╲┘     └╱│╲┘

vs

先剪切，再旋转90°

原始：    剪切后：   旋转后：
┌─┬─┐     ┌╱┬╲┐     ┌─┬─┐
├─┼─┤  →  ├─│─┤  →  ├─┼─┤
└─┴─┘     └╲┴╱┘     └─┴─┘
（结果完全不同！）
```

### 代数表示

```
一般情况下：AB ≠ BA

但注意：(AB)C = A(BC)  （满足结合律）
几何意义：连续三个变换，先结合前两个或后两个，结果相同
```

## 三维情况的推广

### 3×3 矩阵乘法

```
[a b c]   [j k l]
[d e f] × [m n o]
[g h i]   [p q r]

= [aj+bm+cp  ak+bn+cq  al+bo+cr]
  [dj+em+fp  ...       ...      ]
  [...       ...       ...      ]
```

**几何意义完全相同**：每一列是新基向量的位置

### 三维变换的例子

| 变换 | 效果 |
|------|------|
| 绕x轴旋转 | yz平面旋转，x不变 |
| 绕y轴旋转 | xz平面旋转，y不变 |
| 绕z轴旋转 | xy平面旋转，z不变 |
| 缩放 | 三个轴独立缩放 |

## 单位矩阵："什么都不做"

```
    [1 0]           [1 0 0]
I = [0 1]    or    [0 1 0]
                   [0 0 1]

性质：AI = IA = A （对任意矩阵A）
```

**几何意义**：基向量保持原位，空间不变。

## 逆矩阵："撤销变换"

```
如果 A 表示某个变换
则 A⁻¹ 表示"撤销该变换"

A⁻¹A = AA⁻¹ = I
```

**例子**：
- 旋转 θ 度的逆 = 旋转 -θ 度
- 缩放的逆 = 反向缩放

## 关键洞察

### 矩阵乘法的真正含义

> 不要死记硬背乘法公式，记住：**列是变换后的基向量**。

复合变换的列 = 第一个变换作用后，再应用第二个变换。

### 交换律失效的例子

```
旋转 + 剪切：结果依赖于顺序

R = [0 -1]    S = [1 1]
    [1  0]        [0 1]

RS = [0 -1]  ≠  SR = [1  1]
     [1  1]          [-1 0]
```
