# 附注2 - 非方阵

## 核心问题

> 非方阵（m×n，m ≠ n）的几何意义是什么？> > 它们如何将一个维度的空间映射到另一个维度？

## 两种基本情况

### 1. 高 → 低（降维）：m < n

```
3×2 矩阵：将 2D 空间映射到 3D 空间？

不对！应该是 2×3：将 3D 映射到 2D（降维）

输入：ℝ³（三维）──→ 输出：ℝ²（二维）
```

```
例：投影到 xy 平面（忽略 z 坐标）

    [1 0 0]
P = [0 1 0]    (2×3 矩阵)

输入 [x]     输出 [x]
     [y]  →       [y]
     [z]
```

### 2. 低 → 高（嵌入）：m > n

```
3×2 矩阵：将 2D 空间映射到 3D 空间（嵌入）

输入：ℝ²（二维）──→ 输出：ℝ³（三维）
```

```
例：将 xy 平面嵌入到 3D 空间

    [1 0]
E = [0 1]    (3×2 矩阵)
    [0 0]

输入 [x]     输出 [x]
     [y]  →       [y]
                 [0]
                 
结果：3D 空间中的一个平面（z=0）
```

## 非方阵的列视角

### 规则依然适用

> **矩阵的列 = 变换后的基向量位置**

只是现在输入和输出的维度不同！

### 2×3 矩阵例子

```
    │ ↑  ↑  ↑ │
M = │ c₁ c₂ c₃│    (2 行，3 列)
    │ ↓  ↓  ↓ │
    
输入：3D 基向量 î, ĵ, k̂ (每个是 3×1)
输出：2D 向量 (每个是 2×1)

c₁ = M·î = M·[1,0,0]ᵀ  (2×1)
c₂ = M·ĵ = M·[0,1,0]ᵀ  (2×1)
c₃ = M·k̂ = M·[0,0,1]ᵀ  (2×1)
```

**几何**：将三维空间"压扁"到二维平面

### 3×2 矩阵例子

```
    │ ↑  ↑ │
M = │ c₁ c₂│    (3 行，2 列)
    │ ↓  ↓ │
    
输入：2D 基向量 î, ĵ (每个是 2×1)
输出：3D 向量 (每个是 3×1)

c₁ = M·î = 2D→3D
```

**几何**：将二维平面"嵌入"到三维空间

## 秩在非方阵中的意义

### 满秩

```
2×3 矩阵：
• 最大秩 = 2（因为只有2行）
• 满秩 = rank = 2
• 将 3D 压扁到一个"完整"的 2D 平面

3×2 矩阵：
• 最大秩 = 2（因为只有2列）
• 满秩 = rank = 2
• 将 2D 嵌入为 3D 中一个"完整"的平面
```

### 秩亏

```
2×3 矩阵，rank = 1：
• 将 3D 压扁到一条直线

3×2 矩阵，rank = 1：
• 将 2D 嵌入为 3D 中的一条直线
```

## 实际应用

### 数据降维（2×n 矩阵）

```
高维数据：[特征1, 特征2, ..., 特征n]
             ↓
PCA 降维矩阵（2×n）
             ↓
二维可视化：[x, y]
```

### 数据升维（n×2 矩阵）

```
图像：[x, y] 坐标
         ↓
特征映射（n×2）
         ↓
高维特征：[r, g, b, 深度, 纹理, ...]
```

## 关键洞察

### 统一的理解框架

无论方阵还是非方阵：

```
┌─────────────────────────────────────┐
│  矩阵的列 = 变换后基向量的位置      │
│  秩 = 列空间的维度                   │
│  列空间 = 所有可能的输出            │
└─────────────────────────────────────┘
```

唯一区别：
- 方阵：输入输出同维度
- 非方阵：改变维度

### 几何直觉

| 矩阵类型 | 输入 | 输出 | 几何效果 |
|---------|------|------|---------|
| 2×2 | ℝ² | ℝ² | 平面到平面 |
| 2×3 | ℝ³ | ℝ² | 3D→2D（投影）|
| 3×2 | ℝ² | ℝ³ | 2D→3D（嵌入）|
| 3×3 | ℝ³ | ℝ³ | 3D到3D |
