# 07 - 点积与对偶性

## 标准定义 vs 几何直觉

### 代数定义

```
对于向量 v = [v₁, v₂] 和 w = [w₁, w₂]：

v · w = v₁w₁ + v₂w₂

例：[3, 2] · [1, 4] = 3×1 + 2×4 = 11
```

### 几何定义

```
v · w = |v| |w| cos(θ)

其中 θ 是两向量夹角
```

**几何意义**：一个向量在另一个向量方向上的投影长度，乘以被投影向量的长度。

## 点积的几何解释

### 投影视角

```
           w
          ↗
         /|
        / |
       /  | w⊥v (垂直于v的分量)
      /   |
 v ──●────┼────→
    \    /
     \  /  proj_v(w)
      \/
      
v · w = |v| × (w 在 v 上的投影长度)
       = |v| |w| cos(θ)
```

### 正交 = 点积为零

```
当 v ⊥ w 时，θ = 90°，cos(90°) = 0

∴ v · w = 0

几何：w 在 v 上的投影为 0
```

## 对偶性：本节的核心

### 核心洞察

> **每一个向量都对应一个线性变换（1×n 矩阵）**> > **点积 = 应用这个线性变换**

### 具体解释

给定向量 **v** = [a, b]，它可以看作：

**1. 空间中的箭头**

```
     y
     ↑
     │  v=[a,b]
     │ ↗
     ●──→ x
```

**2. 一个线性变换（1×2 矩阵）**

```
v = [a b]  （行向量）

这是一个将 ℝ² → ℝ¹ 的变换（降维）！

[a b] [x] = ax + by = 点积 v·[x,y]
      [y]
```

### 对偶关系

```
向量 v        ⟷      线性变换（对偶向量）
（列向量）          （行向量 = 1×n 矩阵）
                    
    y                    
    ↑  v                 [a b] ——→ ℝ
    │ ↗                   
    ●──→ x                
    
v·w 的值 = 变换(v) 作用于 w 的结果
```

## 投影矩阵与对偶性

### 单位法向量的投影

```
给定单位向量 û，求任意向量 w 在 û 方向的投影：

投影长度 = û · w

这正好是对偶变换的应用！

û = [a, b] （作为行矩阵）
    
投影 = û · w = [a b][w₁] = aw₁ + bw₂
               [w₂]
```

### 几何意义

```
所有与 û 垂直的向量 → 投影为 0
（在零空间中）

û 方向的向量 → 投影 = 原长度
```

## 为什么点积的定义是 "对应分量相乘再相加"？

### 从线性变换推导

```
我们要构造一个线性变换 T: ℝ² → ℝ

T 必须满足：
T([x, y]) = x·T([1,0]) + y·T([0,1])
          = x·a + y·b

所以 T 就是 [a b] 矩阵

而 T([x,y]) = [a b][x] = ax + by = v·w
                  [y]
                  
其中 v = [a, b], w = [x, y]
```

### 关键发现

> **点积的定义 = 线性变换的矩阵乘法形式**

这不是巧合，而是**必然**！

## 对偶空间的直观

### 什么是对偶空间？

> **对偶空间 = 所有线性变换 ℝⁿ → ℝ 的集合**

这些变换可以表示为 1×n 矩阵（行向量）。

### 对偶向量

```
原始空间 ℝ²：向量是列向量 [x]
                                  [y]
                                  
对偶空间 (ℝ²)*：对偶向量是行向量 [a b]

对偶向量作用于向量 = 矩阵乘法 = 点积
```

### 同构关系

```
ℝⁿ ≅ (ℝⁿ)*  （在有限维情况下）

每个向量 ↔ 唯一的对偶向量（线性泛函）

这种对应依赖于内积（点积）的选择
```

## 实际应用

### 1. 计算光照强度（图形学）

```
光照强度 = 法向量 · 光线方向向量

点积直接给出明暗程度！
```

### 2. 判断方向相似度（机器学习）

```
cosine similarity = (v·w) / (|v||w|)

点积标准化后 = 向量夹角的余弦
```

### 3. 功的计算（物理）

```
功 = 力 · 位移 = |F||d|cos(θ)

只有力的投影分量做功
```

## 关键洞察

### 点积的统一视角

```
┌────────────────────────────────────┐
│  点积 v·w                          │
│                                    │
│  = v₁w₁ + v₂w₂ + ...              │ （代数：分量相乘相加）
│                                    │
│  = |v||w|cos(θ)                   │ （几何：投影×长度）
│                                    │
│  = (对偶向量 v)(w)                 │ （对偶性：线性变换）│
└────────────────────────────────────┘
```

### 对偶性的深刻含义

> **向量和线性变换不是完全不同的东西**
> > **通过点积，每个向量都"是"一个线性变换**

这是线性代数中对称性的根源：
- 行向量 vs 列向量
- 向量空间 vs 对偶空间
- 原空间 vs 变换后的空间
