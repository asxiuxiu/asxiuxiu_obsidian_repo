# Self - 个人笔记

这是你自己的地盘。

## 这里放什么？

- ✏️ **手写笔记** - 用自己的话总结的知识
- 💡 **学习思考** - 学习过程中的想法、疑问、顿悟
- 🔧 **实践记录** - 实际动手做过的东西
- 🐛 **问题记录** - 遇到的 bug 和解决方案
- 📝 **原创代码解释** - 自己理解后写的代码注释
- 🎯 **心得体会** - 学习某样东西后的真实感受

## 规则

**AI 不得擅自更改此文件夹中的任何内容。**

除非：
- 你明确要求 AI 帮你修改
- 你要求 AI 帮你排版或格式化
- 其他你明确授权的情况

## 建议

试着把 Generated/ 里的知识，用自己的话重新写一遍放到这里。
只有能用自己的话讲清楚，才算真正理解。
