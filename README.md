# Obsidian 知识库

## 目录结构

```
<vault-root>/
├── Generated/          # 🤖 AI 生成的概念笔记和知识总结
│   ├── AI/             # AI 相关
│   ├── C++/            # C++ 知识
│   ├── GameLearn/      # 游戏开发学习
│   ├── graphics/       # 计算机图形学
│   └── Math/           # 数学
├── Plan/               # 📋 学习计划、方法论、路线图
├── Self/               # ✏️ 个人实践笔记（AI 不得擅自更改）
├── Assets/             # 🖼️ 静态资源（图片等）
├── Workspace/          # 💻 代码实践
└── .agents/            # Agent Skills
```

## 文件夹说明

### Generated/
AI 生成的概念性笔记。这些笔记代表"我了解过这个概念"，但内容大多由 AI 整理生成。

- 可以询问 AI 来生成、修改、补充
- 费曼学习法的结构（Why/What/How）

### Plan/
学习计划、路线图、方法论。

- 学习路径规划
- 打卡记录
- 知识导航

### Self/ ⚠️ 重要
**这是你自己的地盘**。放置：

- 手写笔记
- 学习过程中的思考
- 实践总结
- 原创代码解释
- 问题记录和解决方案

**规则**：除非用户明确要求，AI 不得读取、修改或创建此文件夹中的任何内容。

### Assets/
静态资源，按主题分类存放图片等。

### Workspace/
代码实践区域，由 `workspace-gen` skill 自动生成。

## 使用建议

1. **从 Generated 开始学习** - 快速了解概念
2. **制定 Plan** - 规划学习路径
3. **在 Self 中记录** - 用自己的话写笔记
4. **在 Workspace 实践** - 动手写代码
