# 图形学中的线性代数

> [!info] 前置知识
> 本文假设你已了解基础线性代数概念。如需系统学习线性代数的**几何直观**，请参考 [[Math/线性代数的本质/00-索引|线性代数的本质]] 系列。
>
> 本文重点聚焦**图形学中的应用场景**。

---

## Why：图形学为什么需要线性代数？

在 3D 图形学中，我们需要：
1. **表示位置和方向** — 用向量描述顶点、法线、光线方向
2. **变换物体** — 平移、旋转、缩放都可用矩阵表示
3. **坐标系转换** — 模型空间 → 世界空间 → 相机空间 → 屏幕空间
4. **光照计算** — 点积计算夹角，叉积计算法线

> 没有线性代数，就无法在 2D 屏幕上呈现 3D 世界。

---

## What：核心概念速览

### 向量表示

图形学中，**向量默认为列向量**：

$$
A = \begin{pmatrix}
x \\
y \\
z
\end{pmatrix}
\quad
A^T = \begin{pmatrix}
x & y & z
\end{pmatrix}
$$

### 点乘（Dot Product）

> 详见 [[Math/线性代数的本质/07-点积与对偶性|07-点积与对偶性]]

**代数定义**：
$$
\vec{a} \cdot \vec{b} = x_a x_b + y_a y_b + z_a z_b = |\vec{a}| |\vec{b}| \cos\theta
$$

**图形学应用**：

| 应用 | 说明 | 示例 |
|------|------|------|
| **计算夹角** | 通过 $\cos\theta$ 求角度 | 光源与表面的夹角 |
| **计算投影** | 分解向量到某方向 | 阴影计算 |
| **判断接近度** | 点积越大，方向越接近 | 金属高光判断 |
| **前后判断** | 点积正负判断方向关系 | 面剔除 |

### 叉乘（Cross Product）

> 详见 [[Math/线性代数的本质/08-叉积|08-叉积]]

**代数定义**：
$$
\vec{a} \times \vec{b} = \begin{pmatrix}
y_a z_b - z_a y_b \\
z_a x_b - x_a z_b \\
x_a y_b - y_a x_b
\end{pmatrix}
$$

**关键性质**：
- 结果向量垂直于原向量所在平面
- 右手螺旋定则判断方向
- **反交换律**：$\vec{a} \times \vec{b} = -\vec{b} \times \vec{a}$

**矩阵形式**（将叉乘转化为矩阵乘法）：
$$
\vec{a} \times \vec{b} = A \cdot \vec{b} = \begin{pmatrix}
0 & -z_a & y_a \\
z_a & 0 & -x_a \\
-y_a & x_a & 0
\end{pmatrix} \begin{pmatrix}
x_b \\
y_b \\
z_b
\end{pmatrix}
$$

**图形学应用**：

| 应用 | 说明 |
|------|------|
| **判断左右** | $\vec{a} \times \vec{b}$ 为正 → $\vec{b}$ 在 $\vec{a}$ 左侧 |
| **判断内外** | 叉乘判断点是否在三角形内部（光栅化基础） |
| **构建坐标系** | 用两个向量叉乘得到第三个正交向量 |
| **计算法线** | 三角形两条边叉乘得到面法线 |

### 矩阵与变换

> 详见 [[Math/线性代数的本质/03-矩阵与线性变换|03-矩阵与线性变换]] 和 [[graphics/3. 变换 Transformation|变换]]

**核心性质**：
- 矩阵乘法 **不满足交换律**：$AB \neq BA$
- 矩阵乘法 **满足结合律**：$A(BC) = (AB)C$（这对组合变换很重要）

**常用矩阵**：

| 矩阵 | 说明 |
|------|------|
| **转置** | $(AB)^T = B^T A^T$ |
| **单位矩阵** | $I_{3\times3} = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix}$ |
| **逆矩阵** | $AA^{-1} = A^{-1}A = I$，$(AB)^{-1} = B^{-1}A^{-1}$ |

## How：图形学中的实践

### 1. 坐标系构建

用叉乘从两个向量构建正交坐标系：

```cpp
// 已知：前向向量 forward，近似上向量 up_approx
Vec3 forward = normalize(target - camera_pos);
Vec3 right = normalize(cross(forward, up_approx));  // 叉乘得到右向量
Vec3 up = cross(right, forward);                     // 再次叉乘得到真正的上向量

// 现在 forward, right, up 构成正交基
```

### 2. 点是否在三角形内

光栅化的核心操作：

```cpp
// 判断点 P 是否在三角形 ABC 内
bool point_in_triangle(Vec3 A, Vec3 B, Vec3 C, Vec3 P) {
    Vec3 AB = B - A;
    Vec3 BC = C - B;
    Vec3 CA = A - C;
    
    Vec3 AP = P - A;
    Vec3 BP = P - B;
    Vec3 CP = P - C;
    
    // 三次叉乘，同号说明在内部
    float cross1 = cross(AB, AP).z;  // 假设 2D，取 z 分量
    float cross2 = cross(BC, BP).z;
    float cross3 = cross(CA, CP).z;
    
    return (cross1 > 0 && cross2 > 0 && cross3 > 0) ||
           (cross1 < 0 && cross2 < 0 && cross3 < 0);
}
```

### 3. 向量分解

将向量分解到坐标系各轴：

$$
\vec{p} = (\vec{p} \cdot \vec{x})\vec{x} + (\vec{p} \cdot \vec{y})\vec{y} + (\vec{p} \cdot \vec{z})\vec{z}
$$

其中 $\vec{x}, \vec{y}, \vec{z}$ 是单位正交基向量。
