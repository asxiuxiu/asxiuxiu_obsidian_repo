## 向量运算

### 点乘（Dot Product）

**图形学应用**：
| 应用 | 公式 | 说明 |
|------|------|------|
| **计算夹角** | $\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|}$ | 光源与表面夹角 |
| **判断接近度** | $\vec{a} \cdot \vec{b} > 0$ | 方向是否相近 |
| **前后判断** | $\vec{a} \cdot \vec{b}$ 正负 | 面剔除 |
| **向量投影** | $\text{proj}_{\vec{b}}\vec{a} = \frac{\vec{a} \cdot \vec{b}}{\vec{b} \cdot \vec{b}}\vec{b}$ | 阴影计算 |

**C++ 实现**：
```cpp
float dot(const Vec3& a, const Vec3& b) {
    return a.x * b.x + a.y * b.y + a.z * b.z;
}

// 计算夹角（弧度）
float angle(const Vec3& a, const Vec3& b) {
    return acos(dot(a, b) / (a.length() * b.length()));
}
```

> 点乘的几何意义和推导：[[04-数学基础/07-点积与对偶性]]

### 叉乘（Cross Product）

**图形学核心应用**：

```
        法线 ↑
            │
    B ──────┼────── C
   ╱         │        ╲
  ╱          │         ╲
 ╱           │          ╲
A ─────────────────────────
            
叉乘: AB × AC = 法线方向
```

**关键应用表格**：

| 应用 | 方法 | 代码片段 |
|------|------|---------|
| **计算法线** | 两边叉乘 | `cross(B-A, C-A)` |
| **判断左右** | 叉乘结果符号 | `cross(a, b).z > 0` |
| **判断内外** | 三次叉乘同号 | 见下方代码 |
| **构建坐标系** | 两向量叉乘得第三轴 | `up = cross(right, forward)` |

**C++ 实现**：
```cpp
Vec3 cross(const Vec3& a, const Vec3& b) {
    return Vec3(
        a.y * b.z - a.z * b.y,
        a.z * b.x - a.x * b.z,
        a.x * b.y - a.y * b.x
    );
}

// 判断点 P 是否在三角形 ABC 内（2D 情况）
bool pointInTriangle2D(Vec2 A, Vec2 B, Vec2 C, Vec2 P) {
    Vec2 AB = B - A, BC = C - B, CA = A - C;
    Vec2 AP = P - A, BP = P - B, CP = P - C;
    
    float c1 = cross(AB, AP).z;  // 假设在 xy 平面
    float c2 = cross(BC, BP).z;
    float c3 = cross(CA, CP).z;
    
    return (c1 > 0 && c2 > 0 && c3 > 0) || 
           (c1 < 0 && c2 < 0 && c3 < 0);
}
```

### 向量标准化（Normalization）

几乎所有图形学计算都要求向量是**单位向量**：

```cpp
Vec3 normalize(const Vec3& v) {
    float len = sqrt(v.x*v.x + v.y*v.y + v.z*v.z);
    return Vec3(v.x/len, v.y/len, v.z/len);
}
```

## 矩阵基础

### 列向量约定

图形学中，**向量默认为列向量**，变换矩阵左乘：

```
┌         ┐   ┌   ┐   ┌         ┐
│ a b c │   │ x │   │ ax+by+cz │
│ d e f │ × │ y │ = │ dx+ey+fz │
│ g h i │   │ z │   │ gx+hy+iz │
└         ┘   └   ┘   └         ┘
   M        ×   v    =    v'
```

### 矩阵乘法性质

| 性质 | 说明 | 图形学影响 |
|------|------|-----------|
| **不满足交换律** | $AB \neq BA$ | 变换顺序很重要！先旋转后平移 ≠ 先平移后旋转 |
| **满足结合律** | $A(BC) = (AB)C$ | 组合变换可以按任意顺序计算矩阵 |

**C++ 实现（3×3 矩阵）**：
```cpp
struct Mat3 {
    float m[3][3];
    
    Mat3 operator*(const Mat3& o) const {
        Mat3 r;
        for (int i = 0; i < 3; i++)
            for (int j = 0; j < 3; j++) {
                r.m[i][j] = 0;
                for (int k = 0; k < 3; k++)
                    r.m[i][j] += m[i][k] * o.m[k][j];
            }
        return r;
    }
    
    Vec3 operator*(const Vec3& v) const {
        return Vec3(
            m[0][0]*v.x + m[0][1]*v.y + m[0][2]*v.z,
            m[1][0]*v.x + m[1][1]*v.y + m[1][2]*v.z,
            m[2][0]*v.x + m[2][1]*v.y + m[2][2]*v.z
        );
    }
};
```

## 常用坐标系构建

### 相机坐标系（LookAt）

用叉乘从「前向向量」和「近似上向量」构建正交基：

```cpp
// 输入：相机位置、目标位置、世界向上方向
void buildCameraFrame(Vec3 pos, Vec3 target, Vec3 worldUp,
                      Vec3& forward, Vec3& right, Vec3& up) {
    forward = normalize(target - pos);      // Z轴（指向目标）
    right = normalize(cross(forward, worldUp));  // X轴
    up = cross(right, forward);             // Y轴（真正正交的上方向）
}
```

## 快速参考卡片

### 向量运算速查表

| 运算 | 公式 | 结果类型 |
|------|------|---------|
| 加法 | $\vec{a} + \vec{b}$ | 向量 |
| 减法 | $\vec{a} - \vec{b}$ | 向量 |
| 数乘 | $c\vec{a}$ | 向量 |
| 点乘 | $\vec{a} \cdot \vec{b}$ | **标量** |
| 叉乘 | $\vec{a} \times \vec{b}$ | **向量**（垂直于输入平面）|
| 长度 | $|\vec{a}| = \sqrt{\vec{a} \cdot \vec{a}}$ | 标量 |

### 常用公式

```cpp
// 反射向量（入射方向 i，法线 n）
Vec3 reflect(Vec3 i, Vec3 n) {
    return i - 2 * dot(i, n) * n;
}

// 线性插值（Lerp）
Vec3 lerp(Vec3 a, Vec3 b, float t) {
    return a + (b - a) * t;
}

// 球面线性插值（Slerp，用于旋转）
Vec3 slerp(Vec3 a, Vec3 b, float t) {
    float theta = acos(dot(a, b));
    return (a * sin((1-t)*theta) + b * sin(t*theta)) / sin(theta);
}
```
