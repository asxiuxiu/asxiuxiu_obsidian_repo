# 梯度下降与学习实现

知道了损失函数和数值微分，下一步就是让网络自己学会调整权重。核心思路很简单：**算出损失函数对每个参数的梯度，然后沿着让损失变小的方向更新参数**。这就是梯度下降法。

---

## 梯度下降法

假设损失函数为 $L$，网络参数为 $\mathbf{W}$。我们对每个参数求偏导，得到梯度 $\frac{\partial L}{\partial \mathbf{W}}$。梯度指向损失增长最快的方向，所以沿着梯度的反方向走，就能让损失变小：

$$
\mathbf{W} \leftarrow \mathbf{W} - \eta \frac{\partial L}{\partial \mathbf{W}}
$$

这里的 $\eta$ 叫**学习率**（learning rate），决定每一步迈多大。学习率太大可能跨过最低点，太小又收敛得很慢。

```python
import numpy as np

def gradient_descent(f, init_x, lr=0.01, step_num=100):
    x = init_x.copy()

    for _ in range(step_num):
        grad = numerical_gradient(f, x)
        x -= lr * grad

    return x

def numerical_gradient(f, x):
    h = 1e-4
    grad = np.zeros_like(x)

    it = np.nditer(x, flags=['multi_index'], op_flags=['readwrite'])
    while not it.finished:
        idx = it.multi_index
        tmp_val = x[idx]

        x[idx] = tmp_val + h
        fxh1 = f(x)

        x[idx] = tmp_val - h
        fxh2 = f(x)

        grad[idx] = (fxh1 - fxh2) / (2 * h)
        x[idx] = tmp_val
        it.iternext()

    return grad

# 示例：求 f(x0, x1) = x0^2 + x1^2 的最小值
def function_2(x):
    return x[0] ** 2 + x[1] ** 2

init_x = np.array([-3.0, 4.0])
result = gradient_descent(function_2, init_x, lr=0.1, step_num=100)
print(result)  # 接近 [0. 0.]
```

---

## 学习率的选择

学习率是训练神经网络时最重要的超参数之一：

- **太大**：更新步子太大，可能在最优点附近震荡，甚至发散。
- **太小**：收敛极慢，需要更多迭代次数。

书中的例子很好地说明了这一点：

```python
# 学习率太大：发散
result = gradient_descent(function_2, init_x, lr=10.0, step_num=100)
print(result)  # 变成一个很大的数

# 学习率太小：几乎没动
result = gradient_descent(function_2, init_x, lr=1e-10, step_num=100)
print(result)  # 接近初始值 [-3. 4.]
```

合适的学习率通常需要通过实验来寻找。

## 2 层神经网络类

把参数、预测、损失、梯度计算和参数更新封装成一个类，是组织神经网络代码的标准方式。

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def softmax(a):
    c = np.max(a)
    exp_a = np.exp(a - c)
    return exp_a / np.sum(exp_a)

def cross_entropy_error(y, t):
    if y.ndim == 1:
        t = t.reshape(1, t.size)
        y = y.reshape(1, y.size)

    batch_size = y.shape[0]
    delta = 1e-7
    return -np.sum(t * np.log(y + delta)) / batch_size

class TwoLayerNet:
    def __init__(self, input_size, hidden_size, output_size, weight_init_std=0.01):
        self.params = {}
        self.params['W1'] = weight_init_std * np.random.randn(input_size, hidden_size)
        self.params['b1'] = np.zeros(hidden_size)
        self.params['W2'] = weight_init_std * np.random.randn(hidden_size, output_size)
        self.params['b2'] = np.zeros(output_size)

    def predict(self, x):
        W1, W2 = self.params['W1'], self.params['W2']
        b1, b2 = self.params['b1'], self.params['b2']

        a1 = np.dot(x, W1) + b1
        z1 = sigmoid(a1)
        a2 = np.dot(z1, W2) + b2
        y = softmax(a2)
        return y

    def loss(self, x, t):
        y = self.predict(x)
        return cross_entropy_error(y, t)

    def accuracy(self, x, t):
        y = self.predict(x)
        y = np.argmax(y, axis=1)
        t = np.argmax(t, axis=1)
        accuracy = np.sum(y == t) / float(x.shape[0])
        return accuracy

    def gradient(self, x, t):
        loss_W = lambda W: self.loss(x, t)

        grads = {}
        grads['W1'] = numerical_gradient(loss_W, self.params['W1'])
        grads['b1'] = numerical_gradient(loss_W, self.params['b1'])
        grads['W2'] = numerical_gradient(loss_W, self.params['W2'])
        grads['b2'] = numerical_gradient(loss_W, self.params['b2'])
        return grads

    def params_update(self, grads, learning_rate=0.1):
        for key in ('W1', 'b1', 'W2', 'b2'):
            self.params[key] -= learning_rate * grads[key]
```

> 注意：这里的 `numerical_gradient` 是一个通用函数，和类方法同名容易混淆。书中用这种方式是为了展示原理，实际项目中会用更高效的反向传播来求梯度。

## 训练循环

训练就是反复执行：从训练数据中抽 mini-batch → 计算梯度 → 更新参数。

```python
# 用合成数据演示训练流程
input_size = 2
hidden_size = 3
output_size = 2
network = TwoLayerNet(input_size, hidden_size, output_size)

# 合成训练数据：10 个样本，2 维输入，2 类输出
x_train = np.random.rand(10, input_size)
t_train = np.zeros((10, output_size))
for i in range(10):
    t_train[i, i % output_size] = 1

iters_num = 1000
batch_size = 4
learning_rate = 0.1

train_loss_list = []

for i in range(iters_num):
    # 随机抽取 mini-batch
    batch_mask = np.random.choice(10, batch_size)
    x_batch = x_train[batch_mask]
    t_batch = t_train[batch_mask]

    # 计算梯度并更新参数
    grads = network.gradient(x_batch, t_batch)
    network.params_update(grads, learning_rate)

    # 记录损失
    loss = network.loss(x_batch, t_batch)
    train_loss_list.append(loss)

print(f"最终损失: {train_loss_list[-1]:.4f}")
```

损失逐渐下降，说明网络正在学习。如果画出 `train_loss_list`，通常会看到一条下降的曲线。

## 测试数据评价

训练数据上的表现好不一定代表模型真的好。我们还需要用**测试数据**来评估泛化能力：

```python
# 合成测试数据
x_test = np.random.rand(5, input_size)
t_test = np.zeros((5, output_size))
for i in range(5):
    t_test[i, i % output_size] = 1

train_acc = network.accuracy(x_train, t_train)
test_acc = network.accuracy(x_test, t_test)
print(f"训练准确率: {train_acc}, 测试准确率: {test_acc}")
```

如果训练准确率很高但测试准确率很低，说明模型可能出现了**过拟合**（overfitting），也就是模型记住了训练数据，但没学会通用规律。

---

## 小结

- **梯度下降法**沿着梯度的反方向更新参数，逐步降低损失函数。
- **学习率**控制更新步长，是需要仔细调的超参数。
- 把网络封装成类（如 `TwoLayerNet`），可以更方便地组织参数、损失、梯度和更新。
- **训练循环**的核心是 mini-batch 采样、梯度计算、参数更新。
- 除了看训练损失，还要用**测试数据**评估模型的泛化能力。
