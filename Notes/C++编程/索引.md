```mermaid
    Macro[宏编程] --> CodeGen

    OOP[面向对象设计] --> Poly[多态与虚函数]
    OOP --> Pimpl[Pimpl]
    OOP --> Composition[复合 vs 继承]

    Callable[可调用对象] --> Lambda[Lambda]
    Lambda --> Closure[闭包捕获]
    Callable --> Function[std::function]
    Function --> Delegate[引擎委托系统]

    MemModel[内存模型] --> Atomic[原子操作]
    Atomic --> LockFree[无锁编程]
    MemModel --> MemOrder[内存序]
    LockFree --> ThreadPool[线程池]
    ThreadPool --> WorkSteal[工作窃取]

    Build[编译模型] --> Header[头文件模型]
    Header --> Modules[C++20 Modules]
    Build --> Link[静态/动态链接]
    Link --> ABI[ABI 与跨模块]
    Build --> DebugInfo[调试信息]
```
