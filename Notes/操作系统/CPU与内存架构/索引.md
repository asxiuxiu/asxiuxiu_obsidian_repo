## 阶段目标

1. **建立量化意识**：能说出一次 L1 / L2 / LLC / 主存访问的典型延迟（约 1ns / 4ns / 40ns / 100ns+）。
2. **建立 Cache Line 思维**：任何数据操作的最小单位是 64B，理解这如何影响数据结构设计。
3. **识别 False Sharing**：多线程程序中最隐蔽的性能杀手之一。
4. **理解 MESI**：为后续原子操作与内存序打下基础。
