# 操作系统索引

> 从游戏引擎底层出发的操作系统学习路径。每一块知识都附带可本地编译运行的 C++ 实验，并最终映射回引擎实践。

---

## 学习路线

```
前置基础（建议零基础先读）
├─ 计算机体系结构速览 ──→ CPU、内存、总线、SoC
├─ 编译器与内存布局 ────→ 编译四阶段、堆栈、链接
├─ 操作系统基础 ────────→ 内核态、进程、中断
└─ 网络基础 ────────────→ TCP/IP、Socket、三次握手
       ↓
第一阶段：CPU与内存架构 ──→ 理解 Cache，建立 DOD 思维
       ↓
第二阶段：内存管理 ──────→ 掌握分配器，实现池化内存
       ↓
第三阶段：并发与多线程 ──→ 锁、原子序、无锁编程
       ↓
第四阶段：系统调用与IO ──→ 减少 syscall，优化资源加载
       ↓
第五阶段：性能分析与工具 ─→ 用数据驱动优化，集成 Profiler
```

> [!tip] 零基础学习建议
> 若你发现直接阅读 `CPU缓存体系` 或 `虚拟内存与地址翻译` 有困难（如不理解 CPU 流水线、不知道什么是段错误、不清楚 Socket 的基本流程），强烈建议先读完对应阶段的前置基础笔记。这些笔记补齐了从"零基础"到"操作系统核心概念"之间的断层。
