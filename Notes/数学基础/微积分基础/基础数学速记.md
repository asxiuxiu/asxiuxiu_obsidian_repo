# 基础数学速记

> 一篇覆盖代数核心 + 微积分入门的速查笔记。不求面面俱到，只抓关键概念和公式，用于快速回忆和查漏补缺。

---

## 一、代数核心

### 1. 运算律

所有代数变形的底层只有三条规则：

| 运算律 | 公式 | 关键提醒 |
|--------|------|---------|
| 交换律 | $a+b=b+a$, $ab=ba$ | 减法、除法不能直接用，需转化为加负数/乘倒数 |
| 结合律 | $(a+b)+c=a+(b+c)$ | 添括号注意符号：$a-b-c=a-(b+c)$ |
| 分配律 | $a(b+c)=ab+ac$ | 去括号、提取公因式的核心依据 |

**常见错误纠正**：
- $-(a-b) = -a + b$
- $(a+b)^2 = a^2 + 2ab + b^2$
- $\frac{1}{a} + \frac{1}{b} = \frac{a+b}{ab}$
- $\sqrt{a^2} = |a|$

---

### 2. 方程与不等式

**一元二次方程** $ax^2+bx+c=0$：
- **判别式**：$\Delta = b^2 - 4ac$
  - $\Delta > 0$：两不等实根
  - $\Delta = 0$：一重根
  - $\Delta < 0$：无实根（复数域有两共轭根）
- **求根公式**：$\boxed{x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}}$
- **韦达定理**：$x_1 + x_2 = -\frac{b}{a}$，$x_1 x_2 = \frac{c}{a}$

**不等式**：
- 乘除**负数**时，不等号方向**必须改变**
- 一元二次不等式（设 $a>0$）：
  - $>0$ 取**两边**，$<0$ 取**中间**

---

### 3. 函数本质

> 函数是一种**确定的对应关系**：每个输入 $x$ 对应**唯一**输出 $y$。

**核心性质速查**：

| 性质 | 定义 | 例子 |
|------|------|------|
| 单调性 | $x_1<x_2$ 时，$f(x_1)<f(x_2)$ 为增；反之为减 | 导数 $>0$ 即增，$<0$ 即减 |
| 奇函数 | $f(-x) = -f(x)$，关于原点对称 | $\sin x$, $x^3$, $x$ |
| 偶函数 | $f(-x) = f(x)$，关于 $y$ 轴对称 | $\cos x$, $x^2$ |
| 零点 | $f(x)=0$ 的解 | 图像与 $x$ 轴交点 |

**常见限制**：
- 分母 $\neq 0$
- 偶次根号下 $\geq 0$
- 对数真数 $>0$

---

### 4. 指数与对数

**指数运算律**（$a>0$）：
- $a^m \cdot a^n = a^{m+n}$
- $\frac{a^m}{a^n} = a^{m-n}$
- $(a^m)^n = a^{mn}$
- $a^{-n} = \frac{1}{a^n}$，$a^{\frac{m}{n}} = \sqrt[n]{a^m}$

**对数运算律**（$a>0, a\neq 1, M,N>0$）：
- $\log_a(MN) = \log_a M + \log_a N$
- $\log_a\frac{M}{N} = \log_a M - \log_a N$
- $\log_a M^n = n\log_a M$
- **换底公式**：$\boxed{\log_a b = \frac{\ln b}{\ln a}}$

**自然底 $e$**：$\frac{d}{dx}e^x = e^x$，$\frac{d}{dx}\ln x = \frac{1}{x}$。这就是为什么 $e$ 在微积分中无处不在。

---

### 5. 多项式与因式分解

**乘法公式**（必须熟记）：
- $(a+b)(a-b) = a^2 - b^2$
- $(a\pm b)^2 = a^2 \pm 2ab + b^2$
- $a^3 \pm b^3 = (a\pm b)(a^2 \mp ab + b^2)$
- $(a\pm b)^3 = a^3 \pm 3a^2b + 3ab^2 \pm b^3$

**因式分解口诀**：**一提二套三分组，十字相乘试试看**
1. 提取公因式
2. 套用公式（平方差、完全平方等）
3. 分组分解
4. 十字相乘（二次三项式）

---

### 6. 数列与求和

| 类型 | 通项 | 前 $n$ 项和 |
|------|------|-------------|
| 等差数列 | $a_n = a_1 + (n-1)d$ | $S_n = \frac{n(a_1+a_n)}{2}$ |
| 等比数列 | $a_n = a_1 q^{n-1}$ | $S_n = a_1\frac{1-q^n}{1-q}$ ($q\neq 1$) |
| 无穷等比 | — | $S = \frac{a_1}{1-q}$（$|q|<1$） |

**求和符号**：$\sum_{i=1}^{n} a_i = a_1 + a_2 + \cdots + a_n$
- $\sum i = \frac{n(n+1)}{2}$
- $\sum i^2 = \frac{n(n+1)(2n+1)}{6}$
- $\sum i^3 = \left[\frac{n(n+1)}{2}\right]^2$

## 二、微积分入门

### 7. 导数

> 导数是**变化率**的精确描述：函数在某一点的瞬时变化快慢。

**几何意义**：切线的斜率。

**定义式**：
$$
f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}
$$

**基本求导公式**：

| 函数 | 导数 |
|------|------|
| $c$（常数） | $0$ |
| $x^n$ | $nx^{n-1}$ |
| $e^x$ | $e^x$ |
| $\ln x$ | $\frac{1}{x}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |

**求导法则**：
- **线性**：$(af+bg)' = af' + bg'$
- **乘法**：$(uv)' = u'v + uv'$
- **除法**：$\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$
- **链式法则**：$\boxed{\frac{d}{dx}f(g(x)) = f'(g(x)) \cdot g'(x)}$（复合函数求导核心）

**导数与单调性**：
- $f'(x) > 0$ → 函数递增
- $f'(x) < 0$ → 函数递减
- $f'(x) = 0$ → 可能是极值点

---

### 8. 微分

微分是导数的另一种写法，强调"微小增量"的关系：

$$
dy = f'(x)\,dx
$$

- $dy$：函数值的微小变化
- $dx$：自变量的微小变化
- $f'(x)$：两者之间的比例系数

> 在一元函数中，**可导 $\Leftrightarrow$ 可微**，概念上没有本质区别。微分记号在处理积分换元、微分方程时更方便。

---

### 9. 积分

#### 不定积分

不定积分是求导的逆运算：

$$
\int f(x)\,dx = F(x) + C \quad \text{其中} \quad F'(x) = f(x)
$$

**基本积分公式**：

| 被积函数 | 积分结果 |
|----------|----------|
| $x^n$ ($n \neq -1$) | $\frac{x^{n+1}}{n+1} + C$ |
| $\frac{1}{x}$ | $\ln|x| + C$ |
| $e^x$ | $e^x + C$ |
| $\cos x$ | $\sin x + C$ |
| $\sin x$ | $-\cos x + C$ |

#### 定积分

定积分表示函数曲线与 $x$ 轴之间的**有向面积**：

$$
\int_a^b f(x)\,dx = F(b) - F(a)
$$

这就是**牛顿-莱布尼茨公式**，把面积的计算转化为求原函数在端点的差值。

**定积分性质**：
- 奇函数在对称区间上积分：$\int_{-a}^{a} \text{奇函数}\,dx = 0$
- 偶函数在对称区间上积分：$\int_{-a}^{a} \text{偶函数}\,dx = 2\int_{0}^{a} \text{偶函数}\,dx$

---

### 10. 偏微分

当函数有多个自变量时，$z = f(x, y)$，我们需要区分"对哪个变量求变化率"。

**偏导数**：只对某一变量求导，其他变量视为常数。

$$
\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h, y) - f(x, y)}{h}
$$

**记号**：$\frac{\partial f}{\partial x}$ 也写作 $f_x$。

**几何意义**：在 $y$ 固定时，沿 $x$ 方向的切线斜率。

**梯度（Gradient）**：把所有偏导数拼成一个向量：

$$
\boxed{\nabla f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \right)}
$$

> 梯度指向函数**增长最快**的方向，其模长表示增长速率。这在优化、机器学习、物理场论中极其重要。

## 三、核心脉络图

```
运算律（交换/结合/分配）
      ↓
方程与函数（输入→输出的对应关系）
      ↓
指数/对数/多项式（更高级的运算工具）
      ↓
数列与求和（离散世界的累积）
      ↓
极限（从有限走向无限的关键一跃）
      ↓
导数/微分（瞬时变化率）→ 积分（连续累积）
      ↓
偏微分（多变量世界的变化率）
```

## 四、常见错误速查

| 场景 | 错误 | 正确 |
|------|------|------|
| 指数 | $a^m \cdot a^n = a^{mn}$ | $a^{m+n}$ |
| 对数 | $\log_a(M+N) = \log_a M + \log_a N$ | **不成立** |
| 求导 | $(uv)' = u'v'$ | $u'v + uv'$ |
| 积分 | $\int x^n dx = \frac{x^{n+1}}{n+1}$（$n=-1$） | $\ln|x| + C$ |
| 偏导 | 把其他变量也一起求导 | 只对目标变量求导，其余视为常数 |
