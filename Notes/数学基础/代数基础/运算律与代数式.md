# 运算律与代数式

> 很多人觉得代数"就是算"，但算着算着符号就搞错了、括号就拆错了。其实代数的底层只有三条规则——交换律、结合律、分配律。整式、分式、根式的所有变形，都是从这三条规则长出来的。

## Why：为什么运算律如此重要？

想象你在拼乐高。每一块积木的形状、颜色不同，但连接方式是统一的。运算律就是代数世界的"连接方式"。

**不懂运算律的后果**：
- 去括号时符号总搞错：$-(a - b) = -a - b$（错！）
- 通分时莫名其妙漏乘：$\frac{1}{a} + \frac{1}{b} = \frac{1}{a+b}$（错！）
- 根式运算时乱合并：$\sqrt{a} + \sqrt{b} = \sqrt{a+b}$（错！）

这些问题不是"粗心"，而是对运算律的理解不够扎实。

## What：三条运算律

### 1. 交换律

加法可以换顺序，乘法也可以换顺序：

$$
a + b = b + a, \quad a \times b = b \times a
$$

> 注意：**减法和除法不能交换！** $a - b \neq b - a$，$a \div b \neq b \div a$。
> 但你可以把减法看成"加负数"，把除法看成"乘倒数"，这样就又能交换了。

### 2. 结合律

多个数相加或相乘时，先算哪两个不影响结果：

$$
(a + b) + c = a + (b + c), \quad (a \times b) \times c = a \times (b \times c)
$$

> 这也是为什么 $a - b - c = a - (b + c)$ 成立：
> $a - b - c = a + (-b) + (-c) = a + [(-b) + (-c)] = a - (b + c)$

### 3. 分配律（最重要！）

乘法对加法的分配：

$$
a(b + c) = ab + ac
$$

反过来也常用——**提取公因式**：

$$
ab + ac = a(b + c)
$$

> 去括号、因式分解、合并同类项，本质上都是分配律的正用或反用。

## What：代数式的分类

### 整式

由数和字母通过加减乘组成的式子，分母不含字母。

- **单项式**：$3x^2y$，$-5a$，$7$
- **多项式**：$2x^2 - 3x + 1$（二次三项式）

**同类项**：字母部分完全相同的项。合并同类项就是分配律的反用：

$$
3x^2 + 5x^2 = (3+5)x^2 = 8x^2
$$

### 分式

分母中含有字母的式子，如 $\frac{1}{x}$，$\frac{a+b}{a-b}$。

**基本性质**：分子分母同乘（或同除）一个不为零的整式，分式的值不变：

$$
\frac{A}{B} = \frac{A \times C}{B \times C} \quad (C \neq 0)
$$

**加减法**：先通分，再分子相加减：

$$
\frac{1}{a} + \frac{1}{b} = \frac{b}{ab} + \frac{a}{ab} = \frac{a+b}{ab}
$$

> 切记：$\frac{1}{a} + \frac{1}{b} \neq \frac{1}{a+b}$！这是最常见的错误之一。

### 根式

含有开方运算的式子，如 $\sqrt{2}$，$\sqrt{x}$，$\sqrt[3]{a}$。

**基本性质**：
- $\sqrt{a} \times \sqrt{b} = \sqrt{ab}$ （$a \geq 0, b \geq 0$）
- $\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}$ （$a \geq 0, b > 0$）

**不能乱合并**：
- $\sqrt{a} + \sqrt{b} \neq \sqrt{a+b}$ （除非 $a=b=0$）
- $2\sqrt{3} + 5\sqrt{3} = 7\sqrt{3}$ （这可以，因为是同类二次根式）

## How：常见错误与纠正

| 错误写法 | 正确写法 | 原因 |
|---------|---------|------|
| $-(a-b) = -a - b$ | $-(a-b) = -a + b$ | 负号分配：$-1 \times (-b) = +b$ |
| $(a+b)^2 = a^2 + b^2$ | $(a+b)^2 = a^2 + 2ab + b^2$ | 完全平方公式，中间有交叉项 |
| $\frac{1}{a} + \frac{1}{b} = \frac{1}{a+b}$ | $\frac{1}{a} + \frac{1}{b} = \frac{a+b}{ab}$ | 必须先通分 |
| $\sqrt{a^2} = a$ | $\sqrt{a^2} = |a|$ | 平方根结果非负 |
| $\sqrt{a} + \sqrt{b} = \sqrt{a+b}$ | 不能合并 | 根式加法没有这种分配律 |

## 与高级内容的衔接

- **分配律** → 矩阵乘法对加法的分配、向量内积的线性性质
- **分式通分** → 积分中的部分分式分解
- **根式运算** → 复数模长的运算：$|z_1 z_2| = |z_1| |z_2|$

## 总结

| 概念 | 核心要点 |
|------|---------|
| 交换律 | 加法、乘法可交换；减法、除法需转化为加负数、乘倒数后再看 |
| 结合律 | 加减连续时，添括号要注意符号变化 |
| 分配律 | 去括号、提取公因式的核心依据 |
| 整式 | 分母不含字母；合并同类项是关键操作 |
| 分式 | 通分后再加减；分子分母可同乘同除非零式 |
| 根式 | 乘除可直接运算；加减只能合并同类根式 |

> **代数的所有技巧，本质上都是在玩这三条运算律。当你遇到不确定的变形时，回到交换律、结合律、分配律，问自己：这一步合法吗？**
