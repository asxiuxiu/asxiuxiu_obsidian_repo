# 方程与不等式

> 方程是代数最古老、最核心的应用。从小学就开始解 $x + 3 = 7$，到后来的二次方程、方程组，本质上都是在做同一件事：**通过合法的变形，把未知量孤立出来。**

## Why：为什么要系统学习方程？

**方程的本质**：一个等式，其中包含一个或多个未知量。解方程，就是找出让这些等式成立的未知量的值。

**不懂方程的代价**：
- 看到 $ax^2 + bx + c = 0$ 就发怵，不知道判别式、求根公式怎么用
- 解不等式时，两边乘以一个负数忘记变号
- 遇到实际问题时，无法把文字描述转化为数学表达式

而在更高级的数学中：
- 微分方程是描述自然界变化的方程
- 线性代数是解多元线性方程组的系统理论
- 优化问题常常归结为方程或不等式约束

## What：方程的变形规则

解方程的核心思想是：**等式两边同时进行同样的合法操作，等式仍然成立。**

**合法操作**：
1. 两边同时加上或减去同一个数（或式子）
2. 两边同时乘以或除以同一个**不为零**的数（或式子）

> 为什么强调"不为零"？因为除以零是没有定义的。如果方程两边有公因式，约去前要先判断它是否可能为零。

## What：一元一次方程

形式：$ax + b = 0$ （$a \neq 0$）

解法：

$$
ax = -b \quad \Rightarrow \quad x = -\frac{b}{a}
$$

简单，但它是所有方程解法的原型：**移项 → 合并 → 系数化为 1**。

## What：一元二次方程

形式：$ax^2 + bx + c = 0$ （$a \neq 0$）

### 解法一：因式分解法

如果能把左边拆成两个一次式的乘积：

$$
ax^2 + bx + c = (mx + p)(nx + q) = 0
$$

则 $mx + p = 0$ 或 $nx + q = 0$，分别求解即可。

> 因式分解法最快，但不是所有二次式都能轻松分解。

### 解法二：求根公式（万能）

先计算**判别式**：

$$
\Delta = b^2 - 4ac
$$

- $\Delta > 0$：两个不相等的实数根
- $\Delta = 0$：两个相等的实数根（一个重根）
- $\Delta < 0$：无实数根（在复数域有两个共轭虚根）

求根公式：

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

### 解法三：配方法

把 $ax^2 + bx + c$ 配成完全平方的形式：

$$
ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a}
$$

令其为 0，也能推导出求根公式。配方法在后续学习（如圆锥曲线、二次函数顶点）中非常有用。

### 根与系数的关系（韦达定理）

若 $x_1, x_2$ 是 $ax^2 + bx + c = 0$ 的两个根，则：

$$
x_1 + x_2 = -\frac{b}{a}, \quad x_1 x_2 = \frac{c}{a}
$$

> 韦达定理在不解方程的情况下就能判断根的正负、大小关系，非常实用。

## What：二元一次方程组

形式：

$$
\begin{cases}
a_1x + b_1y = c_1 \\
a_2x + b_2y = c_2
\end{cases}
$$

### 代入消元法

从一个方程中解出一个未知数（如 $x = \ldots$），代入另一个方程，转化为一个一元方程。

### 加减消元法

通过将两个方程同乘某个数，使某一未知数的系数绝对值相等，然后相加或相减消去该未知数。

> 线性代数中的**高斯消元法**，就是加减消元法的系统化、矩阵化版本。

## What：不等式

### 基本性质

不等式与等式的最大区别：**两边同时乘以或除以一个负数时，不等号方向必须改变。**

$$
a > b \quad \Rightarrow \quad -a < -b
$$

### 一元一次不等式

解法与一次方程几乎相同，唯一注意点：**乘除负数要变号。**

### 一元二次不等式

形式：$ax^2 + bx + c > 0$（或 $< 0$，$\geq 0$，$\leq 0$）

**解题步骤**：
1. 先求对应方程 $ax^2 + bx + c = 0$ 的根
2. 根据二次函数图像（抛物线）判断不等式的解集

**口诀**（设 $a > 0$）：
- **大于零取两边**：$ax^2 + bx + c > 0$ 的解集在两个根之外
- **小于零取中间**：$ax^2 + bx + c < 0$ 的解集在两个根之间

## How：解方程的常见错误

| 错误 | 纠正 |
|------|------|
| 两边同除以含未知数的式子，没考虑该式子为零的情况 | 先移项，再提取公因式 |
| 开平方只写正根 | $x^2 = a$ 应写 $x = \pm\sqrt{a}$ |
| 不等式两边乘负数不变号 | 必须变号！ |
| 看到分式方程不解定义域 | 解完后必须验根，排除使分母为零的增根 |

## 与高级内容的衔接

- **二次方程求根公式** → 复数域中的求根、特征值的计算
- **韦达定理** → 多项式根与系数的一般关系、特征多项式
- **方程组求解** → 矩阵求逆、行列式、线性空间
- **不等式** → 微积分中的放缩法、优化问题的约束条件

## 总结

| 类型 | 核心解法 | 关键注意点 |
|------|---------|-----------|
| 一元一次 | 移项、系数化1 | 注意符号 |
| 一元二次 | 因式分解 / 求根公式 / 配方法 | 判别式决定根的情况 |
| 二元一次方程组 | 代入消元 / 加减消元 | 消元后仔细回代 |
| 一元一次不等式 | 同方程，但乘除负数要变号 | 变号！变号！ |
| 一元二次不等式 | 先求根，再看抛物线图像 | 注意开口方向（$a$ 的正负） |

> **解方程就像解谜。你拥有的武器只有三条：等式两边同加减、同乘除（非零）、以及把复杂式子拆成简单式子。所有的技巧，都是这三条武器的变体。**
